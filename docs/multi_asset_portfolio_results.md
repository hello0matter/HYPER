# 多市场一品一策组合首轮结果

时间：2026-07-22。使用 Yahoo 日线（与 Vibe 的 yfinance 数据源同类），默认20个产品，4年历史，手续费与滑点合计按往返20bps，前70%历史选策略，最后30%锁定样本外，合格产品等权。股票、外汇、期货和加密货币的时间戳已经归并成同一个自然日，避免一天被重复计数。

修正后的均衡档结果：20个产品中19个通过选择段。假设本金20,000U：

- 最终金额：约 20,969.60U
- 锁定样本外收益：+4.85%
- 年化收益：+2.77%
- 最大回撤：-6.12%
- Sharpe：0.69
- 同期等权买入持有：+23.44%
- 相对基准：-18.59%

因此问题不是单纯“仓位太保守”，而是策略组合明显跑输最简单的买入持有。激进档（2倍风险、最多8个产品、允许做空、年融资成本8%）在同一窗口约 -32.79%，最大回撤约 -35.57%，同期买入持有约 +42.39%。弱策略加杠杆只会放大弱点。

页面已经加入均衡、进取、激进三个档位，以及风险倍数、最多产品数、融资成本、买入持有基准和逐产品“复制Pine”按钮。TradingView 仍需逐产品复核；它不能在一个 Pine 策略里同时替20个不同图表产品下策略订单。

## V2 滚动组合结果

V2 不再为每个产品永久固定一个历史冠军。最终约30%仍是样本外区，但在其中每约63根日K只使用当时以前约504根日K重新选择；每个产品最多选择3个信号相关性不高于0.75的规则，通过多数投票与 EMA50/EMA200 趋势过滤。候选必须在较早段和验证段都盈利，并在验证段跑赢该产品买入持有，否则该窗口持有现金。跨产品默认使用只依赖过去60根日K波动的风险平价，并限制单产品权重。
